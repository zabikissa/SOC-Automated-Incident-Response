# SOC Incident Response Lab

Simple tuto  SOC pour détection et réponse automatisée d’incidents
avec Splunk, SOAR Phantom et ClamAV.

## Structure
- docs/: Documentation et tutoriels
- splunk/: SPL, alertes, mapping CEF
- soar/: Playbooks Phantom et apps
- endpoint/: Scripts de l’endpoint
- images/: Screenshots et diagrammes
